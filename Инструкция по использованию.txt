1. Запустите программу в Visual Studio Code нажав "Run Python File"
2. В открывшемся меню выберите одну из 5 доступных опций путём ввода цифры от 1 до 5 и нажатием клавиши Enter
3. Если вы выберете "Добавить новый фильм" вам будет предложено ввести название фильма, после чего он будет добавлен и ему будет присвоен ID
4. Если вы нажмете "Показать все доступные фильмы" на экране будет отображен список всех добавленных фильмов с указанием их ID.
5. Если вы нажмете "Добавить нового пользователя" вам будет предложено ввести имя нового пользователя, и ему будет присвоен ID
6. Если вы нажмете "Купить билет" Вам будет предложенно ввести ID пользователя, а затем вам нужно будет ввести ID фильма
7. Если вы нажмете "Отменить покупку билета" вам нужно будет ввести ID билета, если вы введёте неверный ID будет отображено "Билет не найден"
